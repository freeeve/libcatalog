{{- /* Optional cover-art slot. Renders a Work's cover image when the catalog carries one (via the adapter `extra.cover` passthrough), else a graceful lettered placeholder from the title's first character. Gated by the caller on `[params] covers = true`, so a catalog without cover art is unchanged by default. Call with a dict: {"url": , "title": , "variant": "card"|"detail"}. alt is empty because the adjacent title already names the Work (decorative image, WCAG H67) -- avoids a redundant screen-reader announcement; the placeholder is hidden from assistive tech for the same reason. The url is passed through relURL. `lcat export --covers-out` writes covers to /covers/.jpg and the projector emits the site-relative "covers/.jpg", but a src with no leading slash is document-relative in HTML: it resolved against /works// on a detail page and /works/ on the browse list, so the same string 404'd differently on every page. relURL is a no-op on the absolute CDN URLs some deployments carry. */ -}} {{- $url := .url -}} {{- $title := .title | default "" -}} {{- $variant := .variant | default "card" -}} {{- $cls := printf "lcat-cover lcat-cover--%s" $variant -}} {{- if $url -}} {{- else -}} {{- $letter := "?" -}}{{- with $title -}}{{- $letter = substr . 0 1 | upper -}}{{- end -}} {{ $letter }} {{- end -}}
